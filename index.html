📚 EVM Instructions Reference

  • PUSH1 n
    Push 1-byte value onto stack
    PC += 2
  • ADD
    Pop two values, push sum
    PC += 1
  • MUL
    Pop two values, push product
    PC += 1
  • STOP
    Halt program execution
    PC += 1
